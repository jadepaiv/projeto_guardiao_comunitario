# projeto_guardiao_comunitario
O Guardião Comunitário é um sistema que usa inteligência artificial para ajudar a manter as favelas mais seguras. Ele avisa os moradores sobre tiroteios em tempo real, dá dicas de como se proteger e mostra onde encontrar ajuda. Com sensores de som e um aplicativo conectado ao WhatsApp, o sistema empodera a comunidade e ajuda a prevenir a violência.
